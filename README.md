# KinRegistry.ai

## 🎨 FEATURE FOCUS: THE CANVAS CLASSIFIEDS
KinRegistry.ai is primarily a marketplace for **Canvas-based Artworks**. 

### Artist Onboarding
1. **Business Identity:** Artists must provide a valid EIN to register.
2. **Profile Creation:** Each artist receives a 'Depository' page showing their history and available works.
3. **Listing Logic:**
   - Title & Style
   - Canvas Dimensions (H x W x D)
   - Medium (Oil, Acrylic, Mixed Media)
   - Price & Contact Info

### Revenue Model
- **Subscription:** Four tiers (Artisan, Studio, Gallery, Masterpiece).
- **Commission:** Small percentage of sales to support the Solomon Kin Records Foundation efforts.
- **Direct Pay:** Integration with business bank accounts for instant artist payouts.
